# Omni-DM: The Definitive Chronicles

An immersive, AI-powered tabletop RPG experience driven by Google's Gemini models.

## Features

- **Dynamic Storytelling:** Powered by Gemini 3.1 Pro for deep, reactive narratives.
- **Visual Immersion:** Real-time scene visualization using Gemini 2.5 Flash Image.
- **The Chronicle of Souls (Lore System):** A persistent lore book that tracks all your characters and their exploits across different campaigns. Keywords in the narrative are highlighted and clickable for lore pop-ups.
- **Multi-Save Slot System:** Manage multiple campaigns simultaneously with local persistence.
- **Dual Settings:** Choose between "Blood-Sands" (Grimdark Fantasy) and "Chrome-Gut" (Neon-Grim Cyberpunk).
- **Interactive HUD:** Real-time tracking of vitals, stats, inventory, and NPC relationships.

## Getting Started

1. **API Key:** Ensure you have a Gemini API key configured in your environment.
2. **Installation:**
   ```bash
   npm install
   ```
3. **Development:**
   ```bash
   npm run dev
   ```
4. **Build:**
   ```bash
   npm run build
   ```

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **AI:** @google/genai (Gemini 3.1 Pro & Gemini 2.5 Flash Image)
- **Icons:** Lucide React
- **Animations:** Motion (Framer Motion)
- **Markdown:** React Markdown
